**Update 2026 Jul 16**: This post is really outdated, I don't use most of them nowadays.

# System
- [Arch Linux](https://archlinux.org): No proprietary and bloatware, very minimal
- [LineageOS](https://lineageos.org)
- [Tails](https://tails.net), [Whonix](https://whonix.org): Privacy OS with system-wide Tor 

# Browser
- [Librewolf](https://librewolf.net): Firefox but better
- [Tor Browser](https://torproject.org): The best privacy browser but it is super slow
- [uBlock Origin](https://github.com/gorhill/uBlock): Content blocker
- [Multi-Account Containers](https://addons.mozilla.org/en-US/firefox/addon/multi-account-containers): For multi account

# Mail
- [Proton Mail](https://proton.me): Mail provider
- [K-9 Mail](https://k9mail.app): Mail client for Android

# VPN
- [RiseupVPN](https://riseup.net/en/vpn): I use Riseup VPN because it is open-source, free and trustful
- [Orbot](https://orbot.app): Tor on Android

# Search engine
- [DuckDuckGo](https://duckduckgo.com)
- [Startpage](https://startpage.com)

# Password Manager
- [Bitwarden](https://bitwarden.com)
- [KeePassXC](https://keepassxc.org) (i dont use this password manager anymore)

# File sharing
- [Send](https://send,vis.ee): Fork of Firefox Send
- [Syncthing](https://syncthing.net)

# Frontends
- [Newpipe](https://newpipe.net): YouTube frontend for Android
- [Infinity](https://github.com/Docile-Alligator/Infinity-For-Reddit): Reddit frontend for Android
