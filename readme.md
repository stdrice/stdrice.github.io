# stdrice's Personal Site

# Things I use
- Hugo
- PaperMod
- Markdown

# Run
- Install `Hugo`, `git`
- Clone this repo
```
git clone https://github.com/stdrice/stdrice.github.io
```
- Run locally
```
cd stdrice.github.io
hugo server
```
- Create new post
```
hugo new posts/<post-name>/index.md # Post
hugo new vi/<post-name>/index.md # Vietnamese Post
```
