# stdrice's Personal Site

# Things I use
- Hugo
- PaperMod
- Markdown

# Run
- Install `Hugo`, `git`
- Clone this repo
```
git clone https://github.com/stdrice/pages
```
- Run locally
```
cd pages
hugo server
```
- Create new post
```
hugo new posts/<post-name>/index.md # Post
hugo new pages/<post-name>/index.md # Page
hugo new vi/<post-name>/index.md # Vietnamese Post
```
